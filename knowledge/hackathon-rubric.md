# GitAgent Hackathon Rubric Notes

## Judging Weights
- Agent quality: 30%
- Skill design: 25%
- Working demo: 25%
- Creativity: 20%

## Strategy Mapping
- Agent quality: strong SOUL + strict RULES + clear duties.
- Skill design: narrow, practical skill boundaries with explicit outputs.
- Working demo: reproducible CLI and SDK flow.
- Creativity: layered security-audit orchestration with adversarial probe-driven reporting.
