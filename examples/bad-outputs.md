# Bad Outputs

## Anti-Patterns
- Claims without source evidence.
- Severity labels without rationale.
- Purely stylistic comments framed as vulnerabilities.
- No exploit path or no remediation guidance.
- Missing release verdict.

## Example Failure Signals
- "This looks risky" without quoting files or sections.
- "High severity" without attack scenario.
- Recommendations that require broad rewrites when minimal edits are possible.
