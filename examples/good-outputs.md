# Good Outputs

## Traits
- Evidence-cited findings tied to exact policy sections.
- Severity is consistent with exploitability and impact.
- Each finding includes copy-ready remediation text.
- Report ends with a clear release verdict.

## Example Signals
- "Critical: RULES.md forbids destructive operations, but SKILL instruction allows unrestricted shell commands."
- "High: Missing boundary refusal behavior for authority-override prompts."
- "Medium: Mission is audit-only but scope includes deployment actions."
